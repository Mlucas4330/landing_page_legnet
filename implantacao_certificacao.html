Implantação de Certificação ISO

ISO 9001, ISO 14001, ISO 45001 e outras

Conte com a equipe especializada da Legnet para a implantação das principais certificações ISO do mercado como a ISO 9001, ISO 14001, ISO 45001 dentre outras.

ISO 9001

A ISO 9001 é uma norma internacional que estabelece critérios para sistemas de gestão da qualidade.

ISO 14001

A ISO 14001 é a norma internacional que estabelece requisitos para um sistema de gestão ambiental.

ISO 45001

A ISO 45001 é uma norma internacional que define requisitos para um sistema de gestão de Saúde e Segurança do Trabalho (SST).
